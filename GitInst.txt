
1.To make a directory 

mkdir test[Directory name]

2.To open a directory

cd test

3.To initialise a directory 
git init

to commit changes

4.git commit demo.txt -m "committing a text file"

To check the status after commit 

5.git status

To connect with a github account 

6.git config --global user.username pralhad295[Account name on GitHub]

To connect current local repository with a remote repository

7.git remote add origin "link of the remote repository"

to push the committed changes on the remote repository

git push origin master
